Enchancement II

Enchancement 2.1: Job drop down selector

About this enhancement

  • - The first enhancement is “Job position drop down selector”. This enhancement helps user to select their desired job position faster by drop down menu.
  • - By select specific position will hide other position which make users reduce time to scan all job description in job listing page.

Implementation

  • - To implement this enhancement programmer can add more job position by make new section and add new option in select.
  • - This enhancement is the experiment of querySelector() using, by apply on dropdown menu and the selection by CSS selector.
  • - The querySelector() help to reduce amount of code and make code look more clean.

Enchancement 2.2: Job position time limit

About this enhancement

  • - The second enhancement is “Job position time limit”. Every job will have their due date to apply, which is show under position reference number as Deadline: Xd Xh XXm.
  • - When position apply due date is passed the apply for this position will become disable that make that position cannot apply anymore.
  • - This enhancement is on jobs.html, which some jobs are passed due date already and their buttons become disable

Implementation

  • - To implement this enhancement programmer can edit on data-deadline in at start of every job section to adjust that job deadline
  • - This enhancement is implementation of Jquary which is first time to use library in coding and its coundount timer function from W3School to implement this function on page.
  • - JQuary is applied in jobs.html head as CDN link for better responsive.
  • - This enhancement is a good start to know library usage and see different between regular and library code such as way to handle DOM of JQuery by use $ sign make code look more clean and concise
